# Hikari
### A Single Column, High Screen Utilization Wordpress Theme Based on H-Siren.

#### 基于H-Siren二次开发的单栏主题

主要更改部分：

1. 调整栏目宽度，提高屏幕利用率。
2. 添加对无`Gravatar`评论者随机头像功能，头像选取基于邮箱哈希，故相同邮箱总会匹配到同一个随机头像。
3. 替换字体为`XHei Intel(IoF)`
4. 美化主页加载栏动图；调整主页底部高度。
5. 简化页面，移除部分页面提示语句、评论装饰图片。
6. 调整部分提示语句。
7. 移除（屏蔽）了一些功能。
8. 部分细节美化。
